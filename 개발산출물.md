## 서비스 소개

### 서비스 개요

SilverLink는 복지 사각지대에 놓인 어르신들을 위한 AI 음성 복지 도우미입니다.

**문제 인식**:
- 복지 혜택 미신청률 약 30% - 매년 수조원의 복지 예산이 집행되지 못함
- 디지털 소외 - 어르신들이 복잡한 온라인 신청 절차에 막힘
- 정보 부족 - 본인이 받을 수 있는 혜택을 모름

**해결 방법**:
사용자는 음성으로 자신의 상황(나이, 소득, 가구 상황 등)을 설명하면, Google Gemini 2.5 Pro AI가 상황을 분석하여 적합한 복지 혜택을 적합도 점수(0-100점)와 함께 추천합니다. 추천 결과는 텍스트와 음성으로 제공되며, 파일로 다운로드하여 가족과 공유하거나 주민센터 방문 시 활용할 수 있습니다.

**기대 효과**:
- 연간 10만 명 어르신에게 복지 혜택 전달
- 1인당 평균 50만원 × 10만 명 = 500억원 복지 효과
- 디지털 격차 해소 및 복지 접근성 향상

### 타서비스와의 차별점

#### 1. 실시간 녹음 UX 🎙️
**기존 방식**:
```
앱 다운로드 → 회원가입 → 로그인 → 메뉴 찾기 →
녹음 → 파일 저장 → 업로드 → 대기 → 결과 확인
```

**SilverLink**:
```
URL 접속 → 버튼 클릭 → 말하기 → 결과 확인 (30초!)
```

- 파일 업로드 불필요
- 앱 설치 불필요
- 회원가입 불필요
- 브라우저에서 바로 사용

#### 2. Gemini 멀티모달 AI 통합
하나의 AI로 모든 처리:
- STT (음성 → 텍스트)
- 상황 분석 (텍스트 → 이해)
- 복지 매칭 (이해 → 추천)
- 적합도 점수 산출 (추천 → 점수)

#### 3. 즉시 사용 가능한 배포
- Streamlit Cloud 배포 완료
- 모바일 지원 ✅
- 무료 서비스 ✅
- 지금 바로 체험 가능 ✅

#### 4. 할루시네이션 방지
- `welfare_data.json` 기반 검증
- Temperature 0.2로 일관성 제어
- 정확한 복지 정보만 제공
- 2025년 최신 금액 반영 (기초연금 34만 2,510원 등)

### 구현 내용 및 결과물

#### 1. 3가지 입력 방식
- **텍스트 입력**: 직접 타이핑하여 상황 설명
- **실시간 녹음** ⭐: 브라우저에서 바로 녹음 (핵심 차별화)
- **파일 업로드**: MP3/WAV 파일 업로드

#### 2. AI 복지 매칭 시스템
- 20개 복지 혜택 데이터 (`welfare_data.json`)
- Google Gemini 2.5 Pro로 상황 분석
- 적합도 점수 0-100점 산출
- 할루시네이션 방지 검증 시스템
- 2025년 최신 금액 정보 반영

#### 3. 결과 제공
- **텍스트**: 추천 혜택, 대상, 금액, 신청 방법, 필요 서류, 문의처
- **음성 (TTS)**: 한국어 음성으로 친절하게 안내
- **다운로드**: 텍스트 파일, 음성 파일 저장 가능

#### 4. 모바일 최적화
- 반응형 디자인 (768px, 480px 브레이크포인트)
- 터치 최적화
- 큰 글씨, 큰 버튼 (어르신 친화)

### 구현 방식

**기술 스택**:
- **프론트엔드 & 백엔드**: Python 3.11, Streamlit 1.51.0
- **AI**: Google Gemini 2.5 Pro (멀티모달 - STT + 텍스트 생성)
- **TTS**: gTTS (한국어 음성 생성)
- **배포**: Streamlit Cloud (무료)
- **데이터**: JSON 기반 복지 데이터 (20개 혜택)

**아키텍처**:
- 단일 파일 설계 (`app.py` 576줄)
- Streamlit 세션 스테이트 기반 상태 관리
- 탭 기반 UI (3가지 입력 방식)
- 캐싱 (`@st.cache_data`)으로 성능 최적화

**주요 구현 특징**:
1. **Gemini File API**: 오디오 파일을 직접 처리 (별도 STT 불필요)
2. **프롬프트 엔지니어링**: 복지 데이터 전체를 프롬프트에 포함하여 정확한 매칭
3. **JSON 응답 파싱**: AI 응답을 구조화하여 일관된 결과 제공
4. **에러 처리**: API 키, 할당량, 네트워크 오류 등 사용자 친화적 안내

**배포 환경**:
- Streamlit Cloud (https://silverlink-ai-welfare-6kn44w2jypeuce5d9zrsfg.streamlit.app)
- 환경 변수 관리 (Streamlit Secrets)
- HTTPS 지원 (실시간 녹음 필수)
