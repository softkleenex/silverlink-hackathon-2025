## 🤖 AI 기술 활용

### Google Gemini 2.5 Pro (멀티모달 AI)

SilverLink는 단순 API 호출을 넘어 **AI 시스템 설계**에 집중했습니다.

#### 1. 음성 인식 (STT)
- Gemini의 **멀티모달 기능**으로 오디오 직접 처리
- 별도의 STT API 불필요 → 비용 절감 및 통합 처리
- 사투리, 비표준 표현도 이해 가능
- 높은 한국어 인식 정확도

#### 2. AI 복지 매칭
- 20개 복지 혜택 데이터를 프롬프트에 포함
- 사용자 상황 분석 및 **적합도 점수 산출** (0-100점)
- **할루시네이션 방지 시스템**:
  ```python
  # welfare_data.json 기반 검증
  def validate_and_fix_benefits(data):
      welfare_names = [item["name"] for item in welfare_data]
      if benefit_name not in welfare_names:
          # 잘못된 추천 필터링
  ```

#### 3. 응답 일관성 제어
- **Temperature 0.2** 설정으로 응답 일관성 보장
- 같은 입력에 같은 결과 제공
- 신뢰할 수 있는 복지 정보

#### 4. 음성 출력 (TTS)
- gTTS로 한국어 음성 생성
- 어르신들이 듣기 편한 속도와 톤
- MP3 파일로 다운로드 가능

#### 5. 2025년 최신 정보 검증 시스템 ⭐ NEW
- **주요 5개 복지 혜택의 2025년 최신 금액 반영**
  - 기초연금: 월 최대 34만 2,510원 (2024년 대비 인상)
  - 기초생활수급, 주거급여, 노인일자리, 에너지바우처
- **공식 출처 명시**: 보건복지부, 국토교통부 등
- **Few-shot Learning**: 3가지 실제 시나리오 예시로 AI 답변 품질 향상
- **향후 계획**: 복지로 공공 API 연동 (실시간 최신 정보)

**AI 활용 효과**:
- 🎯 정확한 복지 매칭 (적합도 점수 기반)
- 🚀 빠른 처리 속도 (30초 이내)
- 💡 사용자 친화적 경험 (음성 입출력)

## 📋 프로젝트 소개

**SilverLink**는 디지털 소외 어르신들이 받을 수 있는 복지 혜택을 쉽게 찾고 신청할 수 있도록 돕는 AI 음성 서비스입니다.

### 해결하고자 하는 문제
- 복지 혜택 미신청률 약 30% (매년 수조원의 복지 예산 미집행)
- 복잡한 온라인 신청 절차
- 어르신들의 디지털 기기 사용 어려움
- 본인이 받을 수 있는 혜택을 모름

### 핵심 기능
1. 📝 **텍스트 입력**: 키보드로 상황 입력 (가장 간단)
2. 🎙️ **실시간 녹음**: 버튼 클릭으로 바로 녹음 (가장 편리!)
3. 📁 **음성 파일 업로드**: 미리 녹음한 파일 업로드
4. 🤖 **AI 분석**: Google Gemini AI가 상황을 파악하고 적합한 복지 혜택 추천
5. 🔊 **음성 출력**: 추천 결과를 음성으로 친절하게 안내
6. 📥 **결과 다운로드**: 텍스트/음성 파일로 저장 가능

## 📦 설치 방법

### 1. 레포지토리 클론
```bash
git clone <repository-url>
cd ai-conic
```

### 2. 가상환경 생성 및 활성화
```bash
python -m venv venv
source venv/bin/activate  # Mac/Linux
# 또는
venv\Scripts\activate  # Windows
```

### 3. 의존성 설치
```bash
pip install -r requirements.txt
```

### 4. 환경 변수 설정
`.env.example` 파일을 복사하여 `.env` 파일을 생성하고 API 키를 입력하세요.

```bash
cp .env.example .env
```

`.env` 파일 내용:
```
GEMINI_API_KEY=your_gemini_api_key_here
```

## 💡 사용 방법

### 방법 1: 텍스트 입력 (가장 간단)
1. "📝 텍스트 입력" 탭 선택
2. 어르신의 상황을 텍스트로 입력

3. "🔍 복지 혜택 찾기" 버튼 클릭

1. "🎙️ 실시간 녹음" 탭 선택
2. 마이크 버튼 클릭 → 녹음 시작
3. 상황을 말씀하신 후 다시 버튼 클릭 → 녹음 종료

5. 음성으로 결과를 들어보세요

1. "📁 음성 파일" 탭 선택
2. 스마트폰 녹음 앱으로 미리 녹음한 파일 업로드
   - 지원 형식: mp3, wav, m4a
3. AI가 자동으로 분석
4. 결과 확인 및 다운로드

### 📥 결과 활용
- **텍스트 다운로드**: 복지혜택_추천결과.txt
- **음성 다운로드**: 복지혜택_음성안내.mp3
- 가족과 공유하거나 주민센터 방문 시 참고자료로 활용
