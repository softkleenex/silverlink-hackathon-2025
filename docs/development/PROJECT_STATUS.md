## 🎯 프로젝트 개요

### 선택 주제
**C타입**: 복지의 사각지대에 놓인 사람들을 위한 서비스

### 핵심 가치제안
> "디지털 소외 어르신들이 음성만으로 복지 혜택을 찾고 신청할 수 있도록 지원"

- **문제**: 복지 혜택 미신청률 30% (연 수조원 미집행)
- **원인**: 복잡한 온라인 절차, 디지털 격차, 정보 부족
- **해결**: AI 음성 인터페이스로 쉽고 빠르게 복지 정보 제공

- **연간 10만 명** 어르신에게 복지 혜택 전달

## 🔧 기술 스택 요약

### 선택 이유
| 기술 | 선택 이유 | 대안 |
|-----|---------|------|
| **Streamlit** | Python 웹앱 1시간 내 구현 | Flask (복잡), FastAPI (오버스펙) |
| **Gemini API** | 무료, 한국어 우수, 멀티모달 | OpenAI (유료), Claude (비싸) |
| **gTTS** | 완전 무료, 한국어 지원 | Google Cloud TTS (유료) |
| **JSON** | DB 불필요, 빠른 개발 | PostgreSQL (오버엔지니어링) |

### 아키텍처
```
[어르신]
   ↓ (음성/텍스트)
[Streamlit 웹앱]
   ↓
[Gemini API] → STT + AI 분석
   ↓
[gTTS] → 음성 출력
   ↓
[어르신] (결과 확인/다운로드)
```
