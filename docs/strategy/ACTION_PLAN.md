#### 2. AI 활용 강조 (가산점!)

```markdown
## 🤖 AI 기술 활용

### Google Gemini 2.5 Pro (멀티모달 AI)

SilverLink는 단순 API 호출을 넘어 **AI 시스템 설계**에 집중했습니다.

**1. 음성 인식 (STT)**
- Gemini의 멀티모달 기능으로 오디오 직접 처리
- 별도의 STT API 불필요
- 사투리, 비표준 표현 이해 가능

**2. AI 복지 매칭**
- 20개 복지 혜택 데이터를 프롬프트에 포함
- 사용자 상황 분석 및 적합도 점수 산출 (0-100점)
- 할루시네이션 방지 시스템 (welfare_data.json 검증)

**3. 응답 일관성 제어**
- Temperature 0.2 설정
- 같은 입력에 일관된 결과 보장

**4. 음성 출력 (TTS)**
- gTTS로 한국어 음성 생성
- 어르신들이 듣기 편한 속도와 톤
```
