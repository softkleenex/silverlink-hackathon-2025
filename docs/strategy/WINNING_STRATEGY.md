## 🎯 핵심 차별화 포인트 (5분 발표에 집중할 것)

### 1. 실시간 녹음 기능 🎙️ (30초 - 가장 중요!)
**시연 방법**:
- 발표 중 심사위원에게 직접 시연 요청
- "지금 바로 여러분의 상황을 말씀해주시겠습니까?"
- 버튼 클릭 → 녹음 → 즉시 결과 출력 (30초 이내)

**강조점**:
- ✅ 파일 업로드 불필요
- ✅ 어르신도 버튼 한 번으로 사용 가능
- ✅ 해커톤 팀 중 유일한 실시간 녹음 기능

**발표 멘트**:
> "다른 팀들은 파일을 업로드해야 하지만, 저희는 버튼 하나로 끝입니다. 지금 바로 시연해보겠습니다."

### 2. Gemini API 단독 사용 💰 (30초)
**기술적 우위**:
- OpenAI Whisper (STT) + GPT (분석) = 2개 API 필요
- **SilverLink = Gemini 1개로 모두 해결**

**비용 비교**:
| 서비스 | STT 비용 | AI 분석 비용 | 월 예상 비용 (1만 건) |
|--------|---------|------------|---------------------|
| OpenAI | $0.006/min | $0.03/1K tokens | **$600** |
| SilverLink (Gemini) | **$0** | **$0** | **$0** |

**강조점**:
- ✅ 완전 무료 (개발 + 운영)
- ✅ 한국어 특화 (사투리 이해)
- ✅ 음성 파일 직접 이해 → 더 정확

**발표 멘트**:
> "경쟁사들은 월 60만원이 들지만, 저희는 완전 무료입니다. 이것이 바로 지속 가능한 서비스의 시작입니다."

### 3. 모바일 완벽 지원 📱 (20초)
**시연 방법**:
- 발표 중 휴대폰으로 QR 코드 스캔
- 심사위원의 휴대폰으로 직접 테스트 권유

**강조점**:
- ✅ 어르신의 실제 사용 환경 = 휴대폰
- ✅ 터치 최적화 (큰 버튼, 큰 글씨)
- ✅ 데스크톱, 태블릿, 모바일 모두 지원

**발표 멘트**:
> "어르신들은 컴퓨터가 아닌 휴대폰을 사용합니다. 지금 여러분의 휴대폰으로 직접 체험해보세요."
