## 2️⃣ iPhone 미러링 촬영 가이드

### 📱 Part 1: iPhone 미러링 설정 (10분)

#### QuickTime Player 연결 (추천!)

```
1. iPhone을 Mac에 USB 케이블로 연결
2. iPhone: "이 컴퓨터를 신뢰합니까?" → 신뢰
3. Mac: QuickTime Player 실행
4. 파일 → 새로운 동영상 녹화 (⌥⌘N)
5. 녹화 버튼 옆 ▼ 클릭 → iPhone 선택
6. 마이크: iPhone (iPhone 음성 녹음됨)
```

---

### 🎙️ Part 2: 음성 파일 준비 (10분)

#### AirDrop으로 iPhone에 전송

```
Mac에서:
1. Finder → character_voices/ 폴더
2. character_grandma.mp3 우클릭 → 공유 → AirDrop
3. iPhone 선택 → 전송
4. character_senior.mp3도 전송

iPhone에서:
1. 파일 앱 → 다운로드 폴더
2. 2개 MP3 파일 확인
```

#### 🎥 시나리오 1: 독거 할머니 (실시간 녹음) - 50초

**사용 음성**: character_grandma.mp3

**촬영 순서**:

1. **앱 접속 (5초)**
   - Safari에서 배포 URL 접속
   - SilverLink 로고 보이면 2초 정지

2. **탭 이동 (5초)**
   - 화면 스크롤 다운
   - "🎙️ 실시간 녹음" 탭 클릭

3. **녹음 시작 (2초)**
   - "🎙️ 녹음 시작" 버튼 클릭
   - 버튼이 "⏹️ 녹음 종료"로 변경 확인

4. **음성 입력 (7초)**
   - Mac 스피커에서 character_grandma.mp3 재생
   - "저는 72살이고 혼자 살고 있어요. 다리가 아파서 거동이 불편합니다."

5. **녹음 종료 (2초)**
   - "⏹️ 녹음 종료" 버튼 클릭
   - 로딩 화면 2초 대기

6. **AI 분석 대기 (10초)**
   - "🤖 복지 혜택을 찾고 있어요..." 표시
   - 분석 완료까지 대기

7. **결과 화면 (15초)**
   - 결과 화면 천천히 스크롤
   - "👵 어르신 말씀" 섹션 보기
   - 추천 복지 혜택 3개 확인
   - 첫 번째 혜택 "💡 추천 이유" 확장

8. **음성 재생 (4초)**
   - 오디오 플레이어 재생 버튼 클릭
   - 1-2초만 재생 후 일시정지

---

#### 🎥 시나리오 2: 저소득 할아버지 (텍스트 입력) - 50초

**사용 음성**: 없음 (타이핑만)

**촬영 순서**:

1. **탭 이동 (5초)**
   - 화면 맨 위로 스크롤
   - "📝 텍스트 입력" 탭 클릭

2. **텍스트 입력 (15초)**
   - 텍스트 입력 필드 클릭
   - 천천히 타이핑: "68살이고 소득이 적어서 힘들고 일자리를 찾고 싶어요."

3. **검색 실행 (2초)**
   - "🔍 복지 혜택 찾기" 버튼 클릭

4. **AI 분석 대기 (10초)**
   - 로딩 스피너

5. **결과 화면 (13초)**
   - 추천 혜택 3개 확인
   - 첫 번째 혜택 확장

6. **다운로드 (5초)**
   - 화면 아래로 스크롤
   - "📄 결과 텍스트 다운로드" 버튼 보이기
   - "🔊 음성 파일 다운로드" 버튼 보이기
   - (클릭은 안 해도 됨)

#### 🎥 시나리오 3: 건강 문제 (음성 파일 업로드) - 50초

**사용 음성**: character_senior.mp3 (업로드용)

**촬영 순서**:

1. **탭 이동 (5초)**
   - 화면 맨 위로 스크롤
   - "📁 음성 파일" 탭 클릭

2. **파일 선택 (10초)**
   - "Browse files" 버튼 클릭

4. **분석 실행 (2초)**
   - 자동 분석 시작

5. **AI 분석 대기 (10초)**

6. **결과 화면 (15초)**
   - "👵 어르신께서 말씀하신 내용" 확인
   - 추천 혜택 3개
   - 첫 번째 혜택 확장

## 🚨 트러블슈팅

### 녹화가 안 됨
- Mac: 시스템 환경설정 → 보안 및 개인정보보호 → 화면 기록 권한 확인
- QuickTime: iPhone이 목록에 보이는지 확인

### 나레이션 볼륨이 너무 크거나 작음
- 편집 툴에서 오디오 레벨 조정
- 나레이션: -3dB ~ 0dB
- BGM: -18dB ~ -12dB

### 렌더링이 너무 오래 걸림
- 해상도를 1080p → 720p로 낮춤
- 비트레이트 낮춤 (10Mbps → 5Mbps)

### YouTube 업로드 실패
- 파일 크기 확인 (256GB 미만)
- 인터넷 연결 확인
- 다른 브라우저 시도
