## 📱 Part 1: iPhone 미러링 설정 (10분)

### Mac에서 iPhone 미러링

#### 방법 1: QuickTime Player (무선)
```
1. iPhone과 Mac을 같은 Wi-Fi에 연결
2. iPhone: 설정 → 일반 → AirPlay 및 Handoff → AirPlay 수신 켜기
3. Mac: QuickTime Player 실행
4. 파일 → 새로운 동영상 녹화 (⌥⌘N)
5. 녹화 버튼 옆 ▼ 클릭 → iPhone 선택
6. iPhone 화면이 Mac에 미러링됨
```

#### 방법 2: USB 케이블 연결 (안정적, 추천!)
```
1. iPhone을 Mac에 Lightning/USB-C 케이블로 연결
2. iPhone: "이 컴퓨터를 신뢰합니까?" → 신뢰
3. Mac: QuickTime Player 실행
4. 파일 → 새로운 동영상 녹화
5. 녹화 버튼 옆 ▼ 클릭 → iPhone 선택
6. 마이크: iPhone (iPhone 음성 녹음됨)
```

#### 화면 크기 조정
```
QuickTime 윈도우 크기 조정:
- 세로 모드 iPhone 화면이 중앙에 오도록
- 너무 크지 않게 (편집 시 확대 가능)
```

```
- 타이머 시작 (0:00, 0:01, 0:02...)

녹화 팁:
- 버튼 클릭 명확하게
```

```
- 음성 레벨 표시 (있으면)

- 명확하게 들리도록

```
